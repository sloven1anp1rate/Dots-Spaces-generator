# Dots → Spaces generator
### Description
This lightweight tool converts any pasted text by replacing every dot (.) with a space. It’s designed for quick formatting, cleanup, and reversing dot‑separated strings back into readable text. Just paste your content, hit Convert, and the processed output appears instantly. Perfect for filenames, tags, structured identifiers, or undoing dot‑based formatting from other tools.
#
### How does it work?
Same as Spaces → Dot generator except this one replaces spaces with dots!
#
### How to use it?
Insert your doted text inside and click on orange "GENERATE" button, than its going to generate text without dots.
